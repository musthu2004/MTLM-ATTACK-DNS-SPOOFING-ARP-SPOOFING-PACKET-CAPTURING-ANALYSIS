# MTLM-ATTACK-DNS-SPOOFING-ARP-SPOOFING-PACKET-CAPTURING-ANALYSIS
A cybersecurity lab project demonstrating MITM attacks, ARP spoofing, DNS spoofing, packet capturing, and network traffic analysis using Kali Linux, Wireshark, Ettercap, and Bettercap in a controlled environment. 🚀🔐

Project Overview;

This project explores network security concepts through the analysis of ARP Spoofing, DNS Spoofing, and packet-capturing techniques in a controlled laboratory environment. The objective was to understand how network traffic can be intercepted, monitored, and analyzed, while studying the security risks associated with Man-in-the-Middle (MITM) attacks.
The project utilizes industry-standard tools including Tshark, Tcpdump, Ettercap, and Bettercap to examine network communications, capture packets, perform protocol analysis, and demonstrate common network attack vectors for educational and security-awareness purposes.

Conclusion

This project demonstrates how packet-capturing and traffic-analysis tools can be used to identify network vulnerabilities and understand the impact of ARP and DNS-based attacks. The study highlights the importance of secure network configurations, encrypted communications, and continuous monitoring to protect organizational networks from interception attacks.
